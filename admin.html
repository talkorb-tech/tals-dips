פרסם שינויים לאתר

כל השינויים שמורים בדפדפן שלך.
לחצי כדי לעדכן את האתר החי — ישירות.

🔑 הגדרות חיבור ל-GitHub

כדי לפרסם ישירות, נדרש GitHub Token אישי (חד-פעמי):

  1. גשי ל-קישור ליצירת Token
  2. גללי למטה ← לחצי Generate token
  3. העתיקי את הטוקן שמתחיל ב-ghp_
  4. הדביקי אותו כאן:
הטוקן נשמר רק בדפדפן שלך ולא עולה לשום מקום.
